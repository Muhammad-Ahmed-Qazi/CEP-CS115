Create a New Database

  • Steps to create a database:

    • Enter a unique name for your database.
    • Use the 'Add Table' button to add as many input fields as required.
    • Enter names for each table.
    • Use the dropdown button at each table input to show inputs for fields you want in your table.
    • Enter field names and lengths.
    • Use the 'Add field' button to add as many fields as required.
    • Review and submit the form.

Note: Changes to the database are NOT possible after creation!
